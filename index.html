What is CSS?

1.CSS stands for Cascading Style Sheets.

2.CSS describes how HTML elements needs to be displayed on screen,paper or in other media.

3.Css is not neccessaryCSS saves alot of work.It can control the layout of multiple pages all at once.

4.External style sheets are stored in external files.

CSS can be added to HTML documents in 3 ways:

  • Inline-by using style attribute inside HTML elements.
  • Internal-by using style element in head section
  • External-by using link element to link an external CSS file.
  • The most common way to add CSS is to keep the styles in external CSS file. However,in this tutorial we use inline and internal styles.because it is easier to demonstarate and also easier for you to try it yourself.

When comparing CSS class vs ID,The class selector selects HTML elements with a specific class attribute.To select elements with a specific class, write a period (.) character, followed by the class name. The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.

CSS slectors are used to target specific element or a range of elements on a web page. once an element has been targeted then the set of styles has been applied to the particular elements.

There is a wide range of selectors available.Two of the most commonly used are ID and class. Both are used target elements to which styles are to be applied.
